My Favorite Game: Minecraft

Welcome to my mini website! This page is about one of my favorite games, Minecraft. Minecraft is a sandbox game where players can build anything they imagine using blocks. You can explore caves, survive monsters, and create huge worlds.

One cool thing about Minecraft is that every world is randomly generated, which means every time you start a new game the map is completely different.
